Informações básicas

Visando facilitar o acesso à dados abertos do governo para a comunidade de desenvolvedores o Tribunal de Contas da União criou a plataforma AppCívico, que fornece através de uma API de serviços REST informações abertas

Funcionalidades disponíveis

  • Listar escolas
  • Mostrar a média de uma escola pelo seu código
  • Listar os tipos de Avaliações
  • Pesquisar escola por código em um ano
  • Pesquisar escola por código
  • Listar as avaliações por seu tipo
  • Listar avaliações de uma escola
  • Pesquisar Escola pelo nome
  • Pesquisar avaliacao do ano por seu tipo
  • Pesquisar Média da Avaliação por seu ano
  • Pesquisar Avaliação de uma escola por seu ano
